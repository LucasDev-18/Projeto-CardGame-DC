# CardGame DC


## Sobre o projeto
Projeto criado durante o evento MapaDev Week do canal Dev em Dobro (https://www.youtube.com/@DevemDobro)

Uma aplicação com slider que consiste em apresentar cards de alguns personagens do universo DC.
